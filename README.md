# Project Overview

This repository contains code and resources for training Low-Rank Adapters (LoRA) in neural networks. The aim of this project is to provide a simple yet effective approach to enhance the performance of pre-trained models.

## Quick Start Guide

1. **Clone the repository:**  
   ```bash
   git clone https://github.com/Incre15/Master-NS-LoRA-Training.git
   ```  
2. **Navigate to the project directory:**  
   ```bash
   cd Master-NS-LoRA-Training
   ```  
3. **Install the required dependencies:**  
   ```bash
   pip install -r requirements.txt
   ```
4. **Train your model:**  
   ```bash
   python train.py
   ```
